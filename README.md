# liri-node-app

### Overview
This Node module allows users to search with 4 commands:
  * 'concert-this'
    * 'input: an artist'
    * 'search: concerts information.'
  * 'spotify-this-song'
    * 'input: an artist'
    * 'search: song information.'
  * 'movie-this'
    * 'input: a movie'
    * 'search: movie information.'
  * 'do-what-it-says'
    * 'input: random.txt'
    * 'search: functions based on what's defined in random.txt.'
